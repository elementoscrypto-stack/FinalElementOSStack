# ElementOS Vercel Fixed v2

Upload these files at the TOP LEVEL of your GitHub repository:

- index.html
- package.json
- package-lock.json
- vite.config.js
- vercel.json
- src/main.jsx
- src/App.jsx
- src/style.css

Vercel settings:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist
- Root Directory: leave blank unless your files are inside a subfolder
